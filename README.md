# Sistema de Estacionamento

## Descrição
Este projeto é um sistema simples de estacionamento implementado em C#, utilizando Programação Orientada a Objetos (POO). O sistema permite gerenciar veículos estacionados, com as seguintes funcionalidades:
- Adicionar um veículo ao estacionamento.
- Remover um veículo, calculando o tempo estacionado e o valor cobrado.
- Listar todos os veículos atualmente estacionados.

Este projeto foi desenvolvido como parte de um desafio prático de programação para aplicar conceitos básicos de .NET e C#.

## Funcionalidades
- **Adicionar Veículo:** O usuário pode adicionar um veículo fornecendo a placa. A hora de entrada é registrada automaticamente.
- **Remover Veículo:** O usuário pode remover um veículo pela placa. O sistema calcula o tempo que o veículo ficou estacionado e exibe o valor cobrado com base em uma tarifa fixa por hora.
- **Listar Veículos:** Exibe todos os veículos que estão atualmente estacionados, com suas respectivas placas e horários de entrada.
